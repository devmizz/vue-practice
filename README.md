# vue-practice

Vue 찍먹을 해보겠습니다.
일단 [튜토리얼](https://vuejs.org/tutorial/#step-1) 따라하기

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

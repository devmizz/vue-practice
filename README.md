# vue-practice

Vue 찍먹을 해보겠습니다.
1. 일단 [튜토리얼](https://vuejs.org/tutorial/#step-1) 따라하기 ✅
2. 사이드 메뉴 선택에 따라 메인 화면 바꿔보기
3. 계산기 만들어보기 ✅
4. 캘린더 만들어보기
5. 캘린더 고급화

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
